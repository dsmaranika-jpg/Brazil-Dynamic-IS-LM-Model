# Brazil-Dynamic-IS-LM-Model
This project presents a dynamic and interactive IS-LM macroeconomic model for Brazil using Python. The model simulates the effects of fiscal and monetary policy changes on output, interest rates, investment, and equilibrium conditions.

The notebook combines:
Macroeconomic theory
Data visualization
Interactive simulations
Policy analysis
Brazil macroeconomic data

## Dataset
The project uses Brazil macroeconomic data stored in: brazil_macro_data.csv
The dataset contains macroeconomic indicators used for calibration and simulation of the IS-LM framework.
Variables include:
- GDP growth
- Interest rates
- Inflation
- Government expenditure
- Money supply
- Investment indicators
